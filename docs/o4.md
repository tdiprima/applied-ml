## K-Means Clustering 🧩
It's like sorting candies into bowls by color/shape **without knowing colors upfront**.  

- Pick # of bowls (clusters) 👉 e.g., 5 🍬🍭🍫🥜🍒  
- Toss data in → model groups similar ones together 🔄  
- Boom! Patterns emerge instantly 🚀  

**Why perfect for customer behavior segmentation?**  
📊 Thousands of unlabeled records? No problem!  

- **No labels needed** 🤷‍♂️ (unsupervised magic ✨)  
- **Super fast** ⚡ – Fit data in seconds, no training time suck.  
- **Automation gold** 🤖 – Clusters = customer groups → Trigger workflows:  

  | Cluster | Behavior | Action 🎯 |  
  |---------|----------|-----------|  
  | 1 🛒 | Big spenders | VIP emails 💌 |  
  | 2 😴 | Inactive | Wake-up discounts 🔥 |  
  | 3 🚀 | Loyal | Rewards club 🏆 |  

**Result?** Patterns → instant segments → smarter automation! Multiplies your workflows x5 🛠️💥

```py
from sklearn.cluster import KMeans

model = KMeans(n_clusters=5)
model.fit(data)
```
